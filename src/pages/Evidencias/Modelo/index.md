# Ejercicios MER

---

## Ejercicio 1: Hospital – Urgencias

### Enunciado
Crear un diseño E/R que permita gestionar los datos de un Hospital:

- Los pacientes tienen:
  - cédula  
  - dirección  
  - teléfonos de contacto  
  - nombre y apellidos  
  - fecha de nacimiento  
  - edad  

- Los médicos tienen:
  - cédula  
  - nombre completo  
  - especialidad  
  - celular  

- Se debe indicar:
  - si un paciente pasó por Triage  
  - la fecha de la evaluación  
  - el nivel asignado

## Ejercicio 2: Biblioteca

###  Enunciado
Crear un diseño E/R que permita gestionar los datos de una biblioteca de modo que:

- Las personas socias disponen de:
  - código de socio  
  - cédula  
  - dirección  
  - teléfono  
  - nombre y apellidos  

- La biblioteca almacena libros con:
  - título  
  - año de escritura  
  - autor(es)  
  - año de edición  
  - editorial  
  - ISBN  

- Se debe indicar:
  - si un libro está dañado o no
